WorldWise

Frontend 2024

Project Overview

WorldWise is a React-based travel tracking application that allows users to keep track of their adventures around the world. Users can mark cities they've visited on an interactive map, add notes about their experiences, and view their travel history. The application features user authentication and provides a visual representation of travel footprints using an interactive map interface.
