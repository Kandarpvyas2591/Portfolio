Sharable Notes

Frontend 2024

Project Overview

This is a modern, secure, and AI-powered note-taking web application built with React. It allows users to create, edit, pin, encrypt, and search notes, with additional features like grammar checking and AI-generated insights. The app is fully client-side, storing data in the browser for privacy and offline access.

Technologies Used

  • React.js
  • Vite
  • Tailwind CSS
  • Axios
  • Web Crypto API
  • localStorage
  • Groq AI API

Key Features

  • Custom Rich Text Editor
  • Note Management
  • Responsive Design
  • Search
  • Encryption
  • AI Integration: Grammar Check and Insights
  • Persistence
