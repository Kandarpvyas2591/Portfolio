AI Quizzer

Backend 2024

Project Overview

The AI Quizzer is a backend-driven intelligent quiz generation and evaluation system. It enables dynamic quiz creation based on parameters such as grade level, topic, difficulty, and desired score. Once the user submits their answers, the system uses AI to evaluate the responses and generate personalized suggestions for improvement.

Key Features

  • Quiz Generation using AI based on input parameters
  • Answer Evaluation & Feedback with contextual suggestions
  • Email Dispatch of quiz results and insights via Nodemailer and Mailjet
  • Authentication and session security via JWT
  • Efficient performance and caching using Redis
  • Data persistence managed with MongoDB

Technologies Used

  • Node.js
  • Express.js
  • MongoDB
  • Groq API
  • Reddis
  • Nodemailer
  • JWT
